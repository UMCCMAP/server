# 나의 카페 지도 "CMAP"

---
## 문제점 정의
- 꾸준한 커피 시장 규모 증가
- 계속해서 증가하는 커피 매장

**점점 다양한 카페가 많이지는 만큼 사람들은 본인들이 선택해서 카페를 갈 수 있게 되었습니다.**
<br>**하지만 많아진 만큼 가서 실망하는 카페도 많아졌습니다.**
<br></br>**따라서 정리된 카페 정보들과 나만의 카페 지도가 있으면 좋겠다고 생각했습니다.**
